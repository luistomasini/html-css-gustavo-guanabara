Links e âncoras

Usando links externos

Você pode acessar o meu repositório público no Github.

Você também pode acessar o meu canal gratuito no Youtube.

Comentário: É utilizada a tag <a> (que significa âncora) e o parâmetro "href" (hypertext reference) para endereçar o link.
Obs.: Para links externos, utilizar parâmetros target="_blank" e rel="external" ou rel="nofollow" (quando não se dá o aval para o link externo).

Usando links internos

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página

Você também pode acessar a nossa página de notícias

Comentário:

  • parâmetro: href="nome_do_arquivo.html" que está na pasta, ou, href="pasta/nome_do_arquivo.html".
  • parâmetro: target="_self" - mantém a navegação na mesma página.
  • parâmetro: rel="next" e rel="prev" - importante para os mecânismos de busca. Não tem mudança visual.
Obs.: Para voltar páginas, utilizar ./pag.html (mesma página) ou ../pag.html (página anterior)

Links para download

Comentário:

  • Parâmetro href="pasta/nome_do_arquivo.PDF" (ou qualquer outra extensão) define a origem do arquivo no repositório.
  • Parâmetro download="nome_do_arquivo.PDF" (ou qualquer outra extensão) para especificar o arquivo a ser baixado.
  • Parâmetro type (acessar o site https://www.iana.org/assignments/media-types/media-types.xhtml para encontrar os tipos) para dizer ao navegador o tipo de arquivo a ser baixado.
