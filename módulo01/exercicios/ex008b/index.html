Mantendo a formatação do código

Usa-se a tag <pre> antes da tag <code>. Exemplo:

Sem a tag <pre>:

num = int(input('Digite um número')) if num %2 == 0: print(f'0 número {num} é PAR') else: print(f'0 número {num} é ÍMPAR') print('Fim do programa')

Note que os códigos ficarão um do lado do outro, e não é isso que queremos.

Com a tag <pre>:

        
num = int(input('Digite um número'))
if num %2 == 0:
    print(f'0 número {num} é PAR')
else:
    print(f'0 número {num} é ÍMPAR')
print('Fim do programa')

Note que agora sim mantivemos a formatação do código.
